导出错误码说明

导出错误码对照：

错误码详细信息
0正确。没有错误。
-1错误，详情未知
-100导出过程出错
-102编码过程出错
-103解码过程出错
-200Tag级别超过了限定的范围
-201没有选择要导出的合成
-202图片数目超过了限定范围(>=50)
-203webp编码错误
-204导出BMP预合成时渲染截屏错误
-205导出BMP预合成过程出错
-206视频编码过程错误
-207BMP预合成数目太多(>=5)
