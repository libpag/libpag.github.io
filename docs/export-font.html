如何从 AE 中导出字体

如何导出字体

  1. 升级版本：升级 PAGViewer 至最新的 beta 版（1.4.67或以上）。 如何升级至beta版
  2. 配置：打开 AE，点击 “After Effect CC” -> 首选项 -> “PAG Config...”，在弹出窗口中设置 “导出字体”为“是”。如下图：

  3. 导出：正常导出 pag 文件。
  4. 导出 pag 完成后，.pag 文件同目录下将出现若干个 .ttc 后缀的字体库文件。如下图：

说明

  1. 字体文件目录：字体文件存放在 .pag 文件的同目录。
  2. 字体文件数量：pag 文件里包括多少种字体，就会导出多少个字体文件。如果没有文本图层，则不会生成字体文件。
  3. 字体名字：字体文件的文件名(不包括后缀.ttc)即是该字体的名字，也即是 .pag 文件里记录的名字。
  4. 建议：导出字体前，最好先把目标目录下的字体文件删除，避免混淆。
