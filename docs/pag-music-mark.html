  2. 将音频文件添加到目标合成中即可。

在PAGViewer中查看PAG文件结构：getRootLayer：PreComposeLayer -> composition:VectorComposition -> audioBytes 的值，来判断pag文件中音频输入是否导出成功，为 {} 时导出成功，为 null 时导出失败
